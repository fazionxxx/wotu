# 《沃土新生》H5 三消游戏 — 需求文档

> 版本：v1.1  
> 最后更新：2026-06-08  
> 项目用途：2026 年全国土地日主题宣传  
> 载体：H5 网页（微信内置浏览器打开 / 扫码游玩）

---

## 一、项目概述

### 1.1 项目背景

本游戏为 2026 年全国土地日（主题："珍惜每一寸土地 促进高质量发展"）主题宣传活动定制。通过三消（Match-3）玩法结合土地修复策略，让玩家在碎片化娱乐中感知良田培育之不易，树立珍惜土地、守护耕地资源的理念。

### 1.2 产品定位

| 维度 | 说明 |
|------|------|
| 游戏名称 | 《沃土新生》（备选：《耕心》《大地回春》） |
| 游戏类型 | 三消 + 土地修复策略休闲 |
| 载体形式 | H5 网页（微信内打开 / 扫码游玩，无需下载） |
| 目标受众 | 全年龄段，推荐 12 周岁以上 |
| 单局耗时 | 2～5 分钟 / 关 |

### 1.3 宣传目标

- **认知传递**：让用户了解土地退化的常见形式（板结、盐碱、污染、沙化），理解土地修复需循序渐进、科学施策。
- **情感共鸣**：通过粮食储备、步数限制机制，直观感受严守耕地红线的必要性。

---

## 二、故事背景（轻量化）

玩家扮演"土地守护者"，脚下土地因不合理利用出现板结、盐碱、污染、沙化等退化问题。通过三消方式逐步改良土质、修复耕地，在有限步数与粮食储备约束下完成土地复耕任务。每一块复耕完成的良田，都是助力土地高质量发展的具象体现。

## 三、核心玩法机制

### 3.1 方块类型与视觉规范

| 方块名称 | 外观描述 | 属性 |
|----------|----------|------|
| 板结块 | 灰褐色裂纹地块 | 普通污染方块，三消消除 |
| 重金属污染块 | 紫色毒液纹理方块 | 普通污染方块，三消消除 |
| 有机污染物 | 暗绿色腐烂纹理方块 | 普通污染方块，三消消除 |
| 沙化块 | 黄色沙粒纹理方块 | 普通污染方块，三消消除 |
| 盐碱晶 | 白色冰晶块状 | 普通污染方块，三消消除 |
| 疏松土块 | 浅棕色颗粒方块 | 中间过渡形态，三消后升级为健康耕地 |
| 健康耕地 | 深棕底色 + 嫩绿草芽 | 最终成型地块，为通关计数项 |

> **注**：盐碱晶与其他污染物地位完全相同，仅为普通三消方块，无任何特殊击碎机制。

### 3.2 三消规则

#### 3.2.1 普通污染物三消

- **触发条件**：三个相同的污染物在一条直线（横向或纵向）上相邻。
- **消除效果**：
  - 中间格子 → 变为 **1 个疏松土块**
  - 两边格子 → 变为 **空格**
- **后续**：执行重力掉落。

#### 3.2.2 疏松土块三消

- **触发条件**：三个疏松土块在一条直线（横向或纵向）上相邻。
- **消除效果**：
  - 中间格子 → 变为 **1 个健康耕地**
  - 两边格子 → 变为 **空格**
- **计数与增益**：
  - 复耕计数 +1
  - 粮食储备条增加（数值见各关卡设定）
- **后续**：执行重力掉落。

#### 3.2.3 多重匹配处理

一次交换可能同时触发多条直线上的三消，**所有匹配同时消除**，消除完成后统一执行重力掉落。

#### 3.2.4 重力掉落规则

1. 所有空格执行"下落"——上方方块向下掉落填充。
2. 顶部新产生的空位由新方块填充。
3. **新方块类型**：仅限当前关卡配置的污染物种类，**等概率随机**选取。**永不生成疏松土块与健康耕地**。
4. 掉落后若产生新的三消 → 自动继续执行（**连消**），直到棋盘稳定。

#### 3.2.5 交换操作

- 玩家手指拖动**相邻**两个方块完成互换。
- 交换后判定：
  - **有效移动**：产生至少一个三消（普通污染物或疏松土块）。
  - **无效移动**：未产生任何消除效果。
- 每次交换操作（无论有效/无效）均消耗 **1 步**。

### 3.3 胜负判定

| 结果 | 条件 |
|------|------|
| ✅ **胜利** | 累计复耕的健康耕地数量 ≥ 关卡目标值 |
| ❌ **失败** | 步数用完 **或** 粮食储备条归零 |

- 胜利后弹出通关弹窗，随机展示一条习近平耕地保护论述 + 出处。
- 失败后弹出弹窗，说明失败原因（步数耗尽 / 粮食储备归零）。

### 3.4 粮食储备条系统

#### 3.4.1 视觉设计

- **位置**：棋盘顶部，顶部栏中。
- **形态**：水平进度条，内部填充金黄色（代表粮食），图标为麦穗或粮仓。
- **动效**：储备条增加时使用流动的金色粒子表现，增减配进度条过渡动画。
- **头像**：进度条上方放置农民伯伯卡通头像（戴草帽）。
- **命名**："粮食储备"（备选："粮仓满盈"）。

#### 3.4.2 数值规则

| 关卡 | 初始值 | 无效移动扣减 | 成功复耕增益 |
|------|--------|-------------|-------------|
| 第 1 关 | 100% | -10% | +15% |
| 第 2 关 | 60% | -10% | +8% |
| 第 3 关 | 33% | -10% | +3% |

- 储备条归零 → **立即失败**。

### 3.5 步数限制机制

- 每次交换操作消耗 1 步（无论有效/无效）。
- 步数清零且未达标复耕数 → 闯关失败。
- 设计寓意：土地修复窗口期有限，需精打细算合理规划。

## 四、关卡详细设计

### 4.1 第 1 关：复苏之风

| 参数 | 设定 |
|------|------|
| 棋盘规格 | **6×6** |
| 通关目标 | 复耕 **6 块**健康耕地 |
| 总步数 | **30 步** |
| 粮食储备初始 | **100%** |
| 无效移动扣减 | -10% |
| 复耕增益 | +15% |
| 污染物种类 | 3 种：板结块、沙化块、有机污染物（等权重随机） |
| 初始疏松土块 | **3～4 个**，随机分布 |

**设计意图**：新手入门，完整体验「污染物 → 疏松土 → 健康耕地」转化链路，熟悉基础玩法。

## 五、初始棋盘生成规则（通用）

1. 根据关卡参数确定棋盘大小（6×6 / 7×7 / 8×8）与污染物种类列表。
2. 先在棋盘上**随机分布**初始疏松土块（数量见各关卡设定）。
3. 剩余所有格子以**等概率**从当前关卡污染物种类中随机选取填充。
4. 棋盘初始**无空格，全部填满**。
5. **消除初始三消**：检测是否存在三个相同污染物相邻（横/竖方向），若存在：
   - 随机交换一对相邻格子（或局部重新生成），重新检测。
   - 最多尝试 10 次；若仍有三消，手动交换第一对冲突格子打断。

## 六、视觉与交互设计

### 6.1 配色规范

| 元素 | 配色 |
|------|------|
| 主色调 | 大地暖色系（土棕、浅褐） |
| 板结块 | 灰褐色 |
| 重金属污染块 | 紫色 |
| 有机污染物 | 暗绿色 |
| 沙化块 | 土黄色 |
| 盐碱晶 | 白色冰晶 |
| 疏松土块 | 浅棕色 |
| 健康耕地 | 嫩草绿色 |
| 粮食储备条 | 金黄色填充 |

### 6.2 布局结构

#### 顶部栏（所有关卡均显示）

| 显示内容 | 说明 |
|----------|------|
| 当前关卡名称 | 如"复苏之风" |
| 复耕进度 | 格式：`x / 目标值` |
| 剩余步数 | 数字显示，步数紧张时可变色预警 |
| 粮食储备条 | 水平进度条 + 麦穗/粮仓图标 |
| 农民头像 | 进度条上方，戴草帽卡通形象 |

#### 底部按钮

| 按钮 | 显示条件 |
|------|----------|
| 重玩（重置当前关卡） | 始终显示 |
| 下一关 | 通关后出现 |

### 6.3 交互细节

- **操作方式**：手指拖动相邻方块完成互换，适配移动端触屏。
- **特效**：
  - 消除地块附带**震动 + 粒子消散动画**。
  - 粮食储备条增减配**进度条过渡动画（流动金色粒子）**。
- **自适应**：适配屏幕宽度，格子为正方形。
- **禁用操作**：连消动画播放期间锁定操作，动画结束后恢复。

### 6.4 弹窗设计

| 弹窗类型 | 触发条件 | 内容 | 形式 |
|----------|----------|------|------|
| 新手引导 | 第 1 关关键步骤 | 引导提示（如"交换这两个方块试试"；成功后"真棒！你成功复耕了一块土地！"） | **浮动弹窗**（非强制锁定，可跳过），覆盖从首次交换到完成复耕的全流程 |
| 胜利弹窗 | 复耕地块数达标 | 随机展示一条习近平耕地保护论述 + 出处 | 居中弹窗，搭配农田简约插画 |
| 失败弹窗 | 步数耗尽 / 粮食储备归零 | 说明失败原因 | 居中弹窗，搭配土壤简约插画 |

### 6.5 总结画面（三关全通后）

三关全部通关后，展示总结画面，内容包括：
- 总修复耕地数
- 各关成绩回顾
- 主题宣传语
- 可重玩或分享

## 七、音效与音乐

### 7.1 背景音乐

- 全流程播放循环 BGM，风格为温暖、田园、轻快向（契合大地主题）。
- 可静音（提供音效开关按钮）。

### 7.2 音效

| 事件 | 音效 |
|------|------|
| 方块交换 | 轻快滑动音 |
| 污染物三消 | 碎裂 / 消散音 |
| 疏松土块三消（复耕） | 明亮、正向反馈音（如种子破土） |
| 连消 Combo | 音调逐级升高 |
| 无效移动 | 低沉提示音 |
| 粮食储备条变化 | 流动音效 |
| 胜利 | 欢快庆祝音 |
| 失败 | 低沉遗憾音 |

## 八、数据持久化

### 8.1 进度保存

- 使用 `localStorage` 保存当前关卡进度（当前第几关）。
- 用户关闭页面后重新打开 → 恢复到上次所在关卡（重新开始该关卡，不保留棋盘中间状态）。
- 提供"重新开始"入口可重置全部进度。

## 九、通关寄语系统

- **触发时机**：每关通关后胜利弹窗中展示。
- **展示方式**：随机从寄语库中抽取一条，保证同一局内不重复。
- **寄语数量**：预计 12～15 条。
- **内容来源**：习近平总书记关于耕地保护的重要论述及出处。
- **状态**：⏳ 待项目方提供，开发阶段先用占位文本。
